# Student Attendance & Performance Tracker

A web-based system built using PHP and MySQL to manage student attendance and academic performance.

## Features
- Role-based login (Admin, Teacher, Student)
- Admin can manage students
- Teachers can mark attendance and add marks
- Students can view their own records

## Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP

## Database Setup
1. Open phpMyAdmin
2. Create a new database (or just import)
3. Import the file `database.sql`
4. Run the project

## How to Run
1. Install XAMPP
2. Place project in htdocs
3. Import database
4. Start Apache & MySQL
5. Open localhost in browser
